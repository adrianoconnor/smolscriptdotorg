Getting Started

.net 6.0 +

Add the smolscript nuget to your project and then initialize and run your smol scripts in code like this:

{% highlight csharp %} using SmolScript; var exampleSmolScriptCode = @" // This is the smol script program, not .net var a = 10; function multiplyA(valueToMultiplyBy) { a = a * valueToMultiplyBy; } multiplyA(10); "; var vm = SmolVM.Compile(exampleSmolScriptCode); vm.Run(); var valueOfA = vm.GetGlobalVar("a"); {% endhighlight %}
